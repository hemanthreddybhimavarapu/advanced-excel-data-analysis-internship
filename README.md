# Advanced Excel Data Analysis Internship

This repository contains my complete five-week Virtual Advanced Excel Data Analysis Internship work. It documents the full analytics lifecycle from data preparation to strategic optimization using Microsoft Excel.

## Internship Overview
The internship focused on developing practical and analytical skills in advanced Excel, including data cleaning, advanced formulas, dashboard creation, automation planning, and scenario-based decision modeling.

## Key Areas Covered
- Strategic data preparation and cleaning methodologies  
- Advanced Excel formulas (lookup, nested logic, conditional analysis)  
- Dynamic dashboards using Pivot Tables, charts, and slicers  
- Automation strategy using Excel Macros and VBA concepts  
- Scenario analysis and optimization using Goal Seek, Solver, and Data Tables  

## Tools & Skills
- Microsoft Excel (Advanced)  
- Pivot Tables, Charts, Slicers, Conditional Formatting  
- VBA Automation Concepts  
- Business Data Analysis  
- What-If Analysis and Optimization
